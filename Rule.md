### 폴더 규칙

```python
if 스터디 종류 is 모의고사 :
  return "모의고사 이름의 폴더" 
elif 스터디 종류 is 알고리즘 종류 :
  return "알고리즘 이름의 폴더"
else :
  return "날짜와 짧은 설명을 이름으로 갖는 폴더"
```
